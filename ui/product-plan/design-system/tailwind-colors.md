# Tailwind Color Configuration

## Color Choices

- **Primary:** `neutral` — Used for text, nav items, primary UI elements
- **Secondary:** `amber` — Used for active states, hover accents, highlights
- **Neutral:** `stone` — Used for backgrounds, borders, subtle elements

## Usage Examples

```
Primary text: text-neutral-900 dark:text-neutral-100
Active state: text-amber-600 dark:text-amber-400
Background: bg-stone-50 dark:bg-stone-950
Border: border-stone-200 dark:border-stone-800
```
