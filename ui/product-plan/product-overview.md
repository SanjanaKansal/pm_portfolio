# Sanjana's Portfolio — Product Overview

## Summary

A strategic portfolio that serves as a mutual pre-qualification tool and compelling showcase. It helps both parties—employer and candidate—assess compatibility before investing time in a conversation, while demonstrating end-to-end product and technical thinking with precision.

## Planned Sections

1. **Who Am I?** — Your story, background, and the value you bring.
2. **My Toolbox** — Skills, technologies, and capabilities you work with.
3. **Mutual Fit** — Work style, values, and what you're looking for (mutual fit signals).
4. **Contact Me** — Clear call-to-action for aligned visitors to reach out.

## Data Model

**Entities:**
- WhoAmI, ProfileCard, Experience
- MyToolbox, SoftSkill, HardSkill, Project
- MutualFit, Value
- ContactMe, ContactMethod

## Design System

**Colors:**
- Primary: `neutral` — Used for text and nav items
- Secondary: `amber` — Used for active states and hover accents
- Neutral: `stone` — Used for backgrounds and borders

**Typography:**
- Heading: Inter
- Body: Inter
- Mono: IBM Plex Mono

## Implementation Sequence

Build this product in milestones:

1. **Foundation** — Set up design tokens, data model types, and application shell
2. **Who Am I?** — Story, journey, education, and experience grid
3. **My Toolbox** — 4x4 skill grid across Product, AI, Tech, and Soft Skills
4. **Mutual Fit** — What I bring and what I look for
5. **Contact Me** — Identity and social profiles

Each milestone has a dedicated instruction document in `product-plan/instructions/`.
