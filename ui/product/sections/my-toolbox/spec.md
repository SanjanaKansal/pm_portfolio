# My Toolbox

## Overview
A 4x4 grid showcasing skills across four categories: Product, AI, Tech, and User & Soft Skills.

## User Flows
- View all 16 skills organized by category
- Each column represents a skill category with 4 skills

## UI Requirements
- 4-column grid layout with category headers
- Each category has a distinct color accent
- All content visible upfront (no progressive reveal)
